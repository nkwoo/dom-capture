## Возможности

- **Выбор элементов** — Наведите курсор на любой элемент и нажмите для захвата
- **Селектор CSS / XPath** — Захватите конкретный элемент по селектору
- **Захват окна** — Снимок текущей видимой области
- **Захват всей страницы** — Прокручиваемый снимок всей страницы
- **Экспорт** — Сохранить как PNG или PDF или скопировать в буфер обмена

## Установка

### Вариант A — Интернет-магазин Chrome

В [Интернет-магазине Chrome](https://chromewebstore.google.com/detail/dom-capture/cmknlcgmcbinbngoijkmbihpohjkokda) нажмите **Установить**.

### Вариант B — Вручную (Режим разработчика)

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/nkwoo/dom-capture.git
   ```

2. **Открыть страницу расширений Chrome**
   Введите `chrome://extensions` в адресную строку.

3. **Включить Режим разработчика**
   Активируйте переключатель **Режим разработчика** в правом верхнем углу.

4. **Загрузить расширение**
   Нажмите **Загрузить распакованное расширение** и выберите клонированную папку.

5. **Закрепить расширение** (необязательно)
   Нажмите на иконку пазла в панели инструментов Chrome и закрепите **DOM Capture**.

## Использование

| Задача | Как |
|--------|-----|
| Захватить элемент | Нажать на значок расширения → **Выбрать элемент** → Кликнуть на элемент страницы |
| Захватить по селектору | Нажать на значок → Ввести CSS или XPath селектор → **Захватить** |
| Захватить окно | Нажать на значок → Вкладка **Окно** → **Захватить** |
| Захватить всю страницу | Нажать на значок → Вкладка **Вся страница** → **Захватить** |
| Скачать | После захвата нажать **Скачать** (PNG или PDF) |
| Копировать в буфер обмена | После захвата нажать **Копировать** (только PNG) |
